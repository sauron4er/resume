Сайт для роботодавця

Це простий статичний сайт з інформацією про завод та продукти, що він виробляє. Дизайнерів мій роботодавець не залучав, тому вся робота виконана мною від початку і до кінця.

Сайт написаний за допомогою Pug та Bootsrap 4. Позаяк у вимогах не було динамічності, React не використовував зовсім, а Javascript - лише для роботи контактної форми.

Під час роботи над іншим проєктом в мене теж було завдання додати багатомовність. Там використовувався React, тому я підійшов до вирішення інакше: для кожної сторінки створив JSON-файл з перекладами написів, а React підтягував з JSON необхідну для користувача мову.

Уроки

Оскільки була вимога написати сайт на 5 мовах, я зовсім не хотів створювати для кожної мови окрему теку з HTML файлами та вносити правки при необхідності у всі файли. Це, як мінімум, контр-продуктивно. Тому, після невеликого дослідження, обрав варіант написання сайту за допомогою pug. Для кожної сторінки створено один pug-файл, а з нього gulp компілює сторінки для кожної мови. Тож, виконуючи цей проєкт, я навчився працювати з pug і поверхнево ознайомився з gulp.
